发送Ajax请求到您的代理程序

准备工作

  • 1、修改BQIPDAPI.js的第11行，将PROXY_URL变量值修改成您服务器上的正确位置。
  • 2、您应当在应用页面中引用BQIPDAPI.js文件，以便调用其中的方法。

构建请求

	var myUrl="www.mydomain.com";
	var myKey="Keyword1";
	var query=new BQIPDQuery(myUrl,myKey,function(result)
        {
			/*some handle here*/
        }
    );

这样就构建了一个请求！

发送请求

您只要调用BQIPDQuery的Run()方法，就可以实现请求的发送。

	var myUrl="www.mydomain.com";
	var myKey="Keyword1";
	var query=new BQIPDQuery(myUrl,myKey,function(result)
        {
			/*some handle here*/
        }
    );
	query.Run();
